# Špartati
- potomci **Dorcev**
- **vse pravice**
- špartanska vzgoja
- vojaška služba (**falanga**) $\rightarrow$ vojak **hoplit**
# Periojki 
- potomci prislejencev
- osebna **svoboda**
- **brez** političnih pravic
- vojaška služba **lahka pehota** (lokostrelci, pračarji)
# Heloti
- državni **sužnji**
- potomci Ahajcev
